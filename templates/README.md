# 📋 學測英文試卷 Excel 模板

## ⚡ 快速開始

### 1. 安裝依賴並生成模板

```bash
# 安裝必要套件
npm install

# 生成 Excel 模板
npm run generate-exam-template
```

生成的模板位置：`templates/學測英文試卷上傳模板.xlsx`

### 2. 打開模板

使用 Microsoft Excel、Google Sheets 或 LibreOffice 打開生成的模板文件。

### 3. 填寫試卷資料

按照模板中的範例和說明填寫各工作表。

### 4. 上傳至系統

完成填寫後，在後台管理系統上傳 Excel 文件。

## 📁 模板包含的工作表

1. **試卷基本資訊** - 試卷整體資訊
2. **單字題** (10題) - 詞彙選擇題
3. **克漏字組別** - 克漏字文章
4. **克漏字題目** (10題) - 克漏字問題
5. **文意選填組別** - 文意選填文章
6. **文意選填題目** (10題) - 文意選填問題
7. **篇章結構組別** - 篇章結構文章
8. **篇章結構題目** (4題) - 篇章結構問題
9. **閱讀測驗組別** - 閱讀測驗文章
10. **閱讀測驗題目** (12題) - 閱讀測驗問題
11. **混合題組別** - 混合題文章
12. **混合題題目** - 混合題問題
13. **翻譯題** (1-2題) - 中翻英
14. **作文題** (1題) - 英文作文

## ⚠️ 重要提醒

### 必填欄位
所有標示 `*` 的欄位都必須填寫！

### ID 命名規則
```
試卷ID: EXAM_2024_001
組別ID: CLOZE_G1, READ_G2, WORDFILL_G1
```

### 標籤格式
```
單個: 環境
多個: 環境,氣候變遷,科學
```
**使用英文逗號分隔！**

### 答案格式
```
選擇題: A, B, C, D, E (大寫字母)
```
