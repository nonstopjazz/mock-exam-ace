# 學測英文試卷 Excel 模板使用說明

## 🚀 快速開始

### 1. 生成模板

```bash
# 安裝必要套件（如果尚未安裝）
npm install

# 生成 Excel 模板
npm run generate-exam-template
```

模板將會生成在 `templates/學測英文試卷上傳模板.xlsx`

## 📊 工作表說明

### 工作表結構

| 序號 | 工作表名稱 | 說明 | 題型 |
|------|-----------|------|------|
| 1 | 試卷基本資訊 | 試卷的整體資訊 | - |
| 2 | 單字題 | 單字選擇題（10題） | 詞彙 |
| 3-4 | 克漏字組別/題目 | 克漏字填空（10題，2組） | 文法+詞彙 |
| 5-6 | 文意選填組別/題目 | 文意選填（10題） | 文法+詞彙 |
| 7-8 | 篇章結構組別/題目 | 篇章結構（4題） | 邏輯 |
| 9-10 | 閱讀測驗組別/題目 | 閱讀測驗（12題，3組） | 理解 |
| 11-12 | 混合題組別/題目 | 混合題型（不定） | 綜合 |
| 13 | 翻譯題 | 中翻英（1-2題） | 翻譯 |
| 14 | 作文題 | 英文作文（1題） | 寫作 |

## 📝 填寫規範

### 必填欄位
- 所有標示 `*` 的欄位都是**必填**
- 必填欄位不可留空

### 標籤格式
```
單個標籤: 環境
多個標籤: 環境,氣候變遷,科學
```
- 使用**英文逗號**分隔
- 不要有多餘空格

### ID 命名規則

#### 試卷ID
```
格式: EXAM_年份_序號
範例: EXAM_2024_001, EXAM_2024_002
```

#### 組別ID
```
格式: 類型縮寫_G序號
範例:
- CLOZE_G1    (克漏字第1組)
- READ_G2     (閱讀測驗第2組)
- WORDFILL_G1 (文意選填第1組)
- STRUCT_G1   (篇章結構第1組)
- MIXED_G1    (混合題第1組)
```

### 題號規範

| 題型 | 題號範圍 | 範例 |
|------|---------|------|
| 單字題 | 1-10 | 1, 2, 3... |
| 克漏字 | 11-20 | 11, 12, 13... |
| 文意選填 | 21-30 | 21, 22, 23... |
| 篇章結構 | 41-44 | 41, 42, 43, 44 |
| 閱讀測驗 | 45-56 | 45, 46, 47... |
| 混合題 | 57+ | 57, 58, 59... |
| 翻譯題 | T1, T2 | T1, T2 |
| 作文題 | E1 | E1 |

### 答案格式

#### 選擇題
```
單選: A, B, C, D, E
必須使用大寫字母
```

#### 非選擇題
```
翻譯題: 完整英文句子
作文題: 評分標準和範文
```

### 9-10. 閱讀測驗

**組別欄位：**
- 組別ID, 試卷ID, 組別序號, 文章標題
- 文章內容（完整文章，不需標示空格）
- 文章翻譯, 文章類型, 主題標籤

**題目欄位：**
- 題號（45-56）, 組別ID, 題目, 選項A-D
- 正確答案, 詳解, 題型標籤
- Level標籤, 片語標籤, 配分

### 11-12. 混合題

**組別欄位：**
- 組別ID, 試卷ID, 文章標題
- 文章內容, **附加圖表**（圖表描述或URL）
- 文章翻譯, 主題標籤

**題目欄位：**
- 題號（57+）, 組別ID, **題目類型**（選擇/填空/配對/排序）
- 題目, 選項A-D（依題型而定）
- 正確答案, 詳解, 題型標籤

## 🎯 填寫流程建議

### Step 1: 準備工作
1. ✅ 生成模板
2. ✅ 準備試卷PDF或紙本
3. ✅ 準備答案和詳解

### Step 2: 填寫試卷基本資訊
1. ✅ 填寫試卷ID（注意命名規則）
2. ✅ 填寫試卷名稱、年份、月份
3. ✅ 設定難度和總分

### Step 3: 填寫各題型（建議順序）

#### 順序 1: 單字題（最簡單）
- 直接填寫題目、選項、答案、詳解
- 標註Level和主題標籤

#### 順序 2: 閱讀測驗
- 先填寫組別（文章）
- 再填寫題目（對應組別ID）

#### 順序 3: 克漏字
- 先填寫組別（文章，用__1__, __2__標示空格）
- 再填寫題目（空格序號對應）

#### 順序 4: 文意選填
- 先填寫組別（文章+選項列表）
- 再填寫題目（空格對應）

#### 順序 5: 篇章結構
- 先填寫組別（文章+句子選項）
- 再填寫題目（空格對應）

#### 順序 6: 混合題
- 先填寫組別（文章+圖表）
- 再填寫題目（注意題目類型）

#### 順序 7: 翻譯題
- 填寫中文題目和參考答案
- 撰寫評分標準和詳解

#### 順序 8: 作文題
- 填寫作文題目和要求
- 準備範文和寫作提示
- 列出常見錯誤類型

### Step 4: 檢查與驗證
- ✅ 所有必填欄位都已填寫
- ✅ ID格式正確
- ✅ 答案格式統一（大寫字母）
- ✅ 標籤使用英文逗號分隔
- ✅ 組別ID和題目對應正確
- ✅ 題號連續且不重複

## ⚠️ 常見錯誤

### 錯誤 1: ID格式錯誤
```
❌ 錯誤: exam_2024_001, Exam2024001
✅ 正確: EXAM_2024_001
```

### 錯誤 2: 標籤分隔符錯誤
```
❌ 錯誤: 環境；氣候變遷、科學
✅ 正確: 環境,氣候變遷,科學
```

### 錯誤 3: 答案格式錯誤
```
❌ 錯誤: a, option A, 選項A
✅ 正確: A
```

### 錯誤 4: 空格標示錯誤
```
❌ 錯誤: In recent years _1_..., In recent years [1]...
✅ 正確: In recent years __1__...
```

### 錯誤 5: 選項列表格式錯誤
```
❌ 錯誤: A)word B)word, A.word B.word
✅ 正確: (A)word (B)word (C)word...
```

## 💡 進階技巧

### 使用 Excel 資料驗證

#### 設定答案下拉選單
1. 選擇「正確答案」欄位
2. 資料 → 資料驗證
3. 允許：清單
4. 來源：`A,B,C,D,E`

#### 設定難度下拉選單
1. 選擇「難度」欄位
2. 資料 → 資料驗證
3. 允許：清單
4. 來源：`簡單,中等,困難`

### 使用條件格式標示必填

1. 選擇必填欄位範圍
2. 常用 → 條件格式 → 新增規則
3. 規則類型：只格式化包含以下內容的儲存格
4. 格式化只包含：空白
5. 設定格式：紅色底色
